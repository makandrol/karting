# Розклад змагань — Картодром "Жага швидкості"

## Тижневий розклад

| День | Час | Змагання | Формат |
|------|-----|----------|--------|
| Понеділок | 20:00 | Гонзалес | 12-24 раунди × 2 кола |
| Вівторок | 19:30 | Лайт Ліга | Квала + 2 гонки |
| Середа | 20:00 | Ліга Чемпіонів | Квала + 3 гонки |

## Авто-визначення

Collector автоматично визначає початок змагань:

1. **Перевіряє день тижня + час** (вікно ±30 хвилин)
2. **Визначає тренувальні заїзди** (5 хв, 1-2 сесії)
3. **Чекає перерву** (3-15 хвилин — зміна траси, розподіл по картах)
4. **Позначає наступні заїзди** як фази змагання

### Сигнали перерви:
- Ніхто не їздить або 1 пілот (тестує карти)
- Тривалість 3-15 хвилин

## Ручне управління

Адміни можуть:
- **Запустити** змагання вручну (відміняє автовизначення на цей день)
- **Зупинити** змагання
- **Позначити** конкретний заїзд як фазу (Квала / Гонка N)
- **Скинути** автовизначення

## API

```
GET  /competition              — поточний стан
POST /competition/start        — вручну запустити {format, name}
POST /competition/stop         — зупинити
POST /competition/phase        — позначити фазу {sessionId, type, name}
POST /competition/reset        — скинути
```
