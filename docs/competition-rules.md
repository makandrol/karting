# Регламент змагань

## Гонзалес

- **Карти**: 12 картів
- **Пілоти**: 12–24
- **Формат**: Кожний пілот проїжджає на кожній з 12 картів по 2 залікових кола
- **Залік**: Враховується швидше з 2 кіл для кожного карту
- **Рейтинг**: Пілоти сортуються по середньому часу серед найшвидших кіл для кожного карту
- **Етапи**: 12 раундів (по 1 на кожний карт), кожен раунд = 1 заїзд

## Лайт Ліга

- **Макс. пілотів**: 36 (після кваліфікації)
- **Групи**: <=13 → 1 група, 14–26 → 2 групи, 27+ → 3 групи
  - При нерівному розподілі більше пілотів в перших (сильніших) групах

### Етапи

1. **Кваліфікація** — на випадковому карті. Визначає порядок і розподіл по групах.
2. **Гонка 1** — Їдуть по групах: спочатку 3-тя (25–36), потім 2-га (13–24), потім 1-ша (1–12).
   Стартовий порядок реверсивний (останній у групі стартує першим).
3. **Гонка 2** — Пілоти сортуються по найшвидшому колу під час Гонки 1, гонка в такому ж форматі.

### Бали

- **Топ-5 найшвидших** у кваліфікації та кожній гонці: 2.5, 2.0, 1.5, 1.0, 0.5 балів
- Бали за позиції та обгони (деталі будуть описані окремо)
