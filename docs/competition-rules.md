# Регламент змагань

## Гонзалес

- **Карти**: 12 картів
- **Пілоти**: 12–24
- **Формат**: Кожний пілот проїжджає на кожній з 12 картів по 2 залікових кола
- **Залік**: Враховується швидше з 2 кіл для кожного карту
- **Рейтинг**: Пілоти сортуються по середньому часу серед найшвидших кіл для кожного карту
- **Етапи**: 12 раундів (по 1 на кожний карт), кожен раунд = 1 заїзд

## Лайт Ліга

- **Макс. пілотів**: 36 (після кваліфікації)
- **Групи**: <=13 → 1 група, 14–26 → 2 групи, 27+ → 3 групи
  - При нерівному розподілі більше пілотів в перших (сильніших) групах

### Етапи

1. **Кваліфікація** — на випадковому карті. Визначає порядок і розподіл по групах.
2. **Гонка 1** — Їдуть по групах: спочатку 3-тя (25–36), потім 2-га (13–24), потім 1-ша (1–12).
   Стартовий порядок реверсивний (останній у групі стартує першим).
3. **Гонка 2** — Пілоти сортуються по найшвидшому колу під час Гонки 1, гонка в такому ж форматі.

### Бали

- **Топ-5 найшвидших** у кваліфікації та кожній гонці: 2.5, 2.0, 1.5, 1.0, 0.5 балів
- Бали за позиції та обгони (деталі будуть описані окремо)

## Спринт

- **Макс. пілотів**: 45
- **Групи**: ≤14 → 1 група, 15–29 → 2 групи, 30+ → 3 групи
- **Кількість пілотів** визначається по Квалі 1 (та сама логіка, що ЛЛ/ЛЧ)

### Етапи

1. **Кваліфікація 1** (по групах) — визначає стартові позиції для Гонки 1
2. **Гонка 1** (по групах) — реверсивний старт від квалі 1
3. **Кваліфікація 2** (по групах) — визначає стартові позиції для Гонки 2
4. **Гонка 2** (по групах) — реверсивний старт від квалі 2
5. **Фінал** (по групах) — стартовий порядок за сумою балів Гонки 1 + Гонки 2

### Розбиття на групи

- **Гонки 1, 2**: snake/round-robin розбиття (`splitIntoGroupsSprint`) — збалансовані групи
- **Фінал**: послідовне ярусне розбиття (Sequential tiered split):
  - Найкращі → Pro (група 1)
  - Середні → Gold (група 2)
  - Решта → Light (група 3)
  - Сортування за сумою балів гонок 1+2 (tiebreak: найкращий час квалі 1)

### Бали

- **Без балів за обгони** — тільки за позиції + швидкість
- **Бали за позицію** у гонках 1, 2: 40 / 37 / 35 / 33 / 31 / 29 / 27 / ... (-2 за кожну наступну позицію, мін. 0)
  - Функція: `getSprintPositionPoints(finishPos)`
- **Бали за позицію у фіналі**: безперервна шкала від 180, -3 за кожну позицію через всі групи
  - Pro отримує найбільше, потім Gold, потім Light
  - Функція: `getSprintFinalPoints(finishPos, precedingPilots)` — `precedingPilots` = кількість пілотів у вищих групах
- **Швидкісні бали**: 1 бал за найшвидше коло в кожній групі — в кожній кваліфікації та гонці
- **Загальний рейтинг**: сума швидкісних балів квалі 1 + квалі 2 + всі бали гонок 1, 2 та фіналу

### Таблиця результатів (LeagueResults)

- Колонки: Кв1 (карт, час, швидк.), Г1 (група, старт, фініш, карт, час, швидк., штрафи, позиція, сума), Кв2, Г2, Фінал, Сума
- Порядок у "Бали": Швидк, Штрафи, Позиція, Сума
- Г2 Сума = кумулятивна (q1_speed + r1_total + q2_speed + r2_total)
- Фінал Сума = totalPoints (загальна сума змагання)
- Сортування: Сума, Кв1, Г1, Кв2, Г2, Г2 сума, Фінал
